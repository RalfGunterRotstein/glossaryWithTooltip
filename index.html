Glossary With Tooltips

This is a minimalist example of a jQuery script and some CSS styles that, together, highlight SPANS containing a “data-keyword” property, displaying their definition on hover, click or touch, and insert in the page a glossary with all the keywords and definitions in alphabetical order. Click the keywords to see their definitions. Requires a script that adds the class “click” to clicked elements with the class “clickable”. I’ve used my own. (Code available on GitHub.)
